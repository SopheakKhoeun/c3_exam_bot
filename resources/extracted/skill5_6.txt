Part  IV  —  Case  Study  (35  Marks)  
Employee  Management  System  
A  company  wants  to  develop  an  Employee  Management  System  to  store  employee  
information
 
and
 
department
 
information.
 
Requirements  
Create  a  database  named  CompanyDB .  
The  database  contains  the  following  tables.  
Departments  
●  department_id  ●  department_name  ●  location  
Employees  
●  employee_id  ●  full_name  ●  gender  ●  date_of_birth  ●  phone_number  ●  salary  ●  department_id  
Each  employee  belongs  to  one  department ,  while  one  department  can  have  many  
employees
.
 
 
Tasks  
1.  Write  an  SQL  script  to  create  the  database  CompanyDB.  (3  marks)   
2.  Write  SQL  scripts  to  create  both  tables.  (3  marks)   
3.  Select  appropriate  data  types  for  each  field.  (3  marks)   
4.  Define  the  Primary  Key  and  Foreign  Key  for  each  table.  (5  marks)   
5.  Write  SQL  scripts  to  insert  at  least  5  records  into  each  table.  (5  marks)   
6.  Write  SQL  queries  to:  
●  Display  all  employees.  (2  marks)  ●  Display  employees  whose  salary  is  greater  than  $500 .  (3  marks)  ●  Display  employee  names  together  with  their  department  names.  (5  marks)  ●  Display  the  number  of  employees  in  each  department.  (5  marks)  
 
7.  Write  an  SQL  statement  to  update  the  phone  number  of  one  employee.  (4  
marks)
  
8.  Write  an  SQL  statement  to  delete  one  employee  record  from  the  
database.
 
(5
 
marks)
